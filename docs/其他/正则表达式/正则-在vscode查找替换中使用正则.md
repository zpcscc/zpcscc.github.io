## 正则-在vscode查找替换中使用正则

### 1、打开vscode查找替换窗口

### 2、按下查找窗口右边匹配全部的按钮

​	vscode查找窗口有三个基础匹配规则，这里我们选择第三个

```
1、是否启用，匹配大小写
2、是否启用、字符完全匹配，不进行模糊匹配
3、是否启用正则表达式搜索
```

### 3、在搜索窗口直接输入正则表达式即可，不用输入正则的斜杠和全局等修饰，vacode默认全局

```typescript
例如：
匹配数字的正则是：/^[0-9]*$/
我们直接在搜索框输入: [0-9]
```
